# Mojic - 動作確認手順書

## 📱 初回起動手順

### 1. 環境準備
```bash
cd rapid-type
npm install  # 依存関係インストール（済み）
```

### 2. アプリ起動
```bash
npx expo start
```

### 3. テスト方法
- **iOS シミュレータ**: `i` キーを押す
- **Expo Go アプリ**: QRコードをスキャン（iOS実機推奨）

## 🐛 既知の問題・注意点

1. **Expo Go での制限**
   - 一部のネイティブ機能は Expo Go では動作しない場合あり
   - 本番ビルドで再テスト推奨

2. **サウンド**
   - 現在プレースホルダー（実際の音声ファイル未実装）
   - 本番では実際の音声ファイルを追加

3. **広告**
   - テストモードのみ（実際のAdMob広告は本番で有効化）

## 📋 デバッグコマンド

```bash
# TypeScript チェック
npx tsc --noEmit

# Metro バンドラーキャッシュクリア
npx expo start --clear

# 依存関係の確認
npm ls

# Expo Doctor
npx expo-doctor
```
