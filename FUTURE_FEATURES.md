# FUTURE_FEATURES.md - Post-MVP Features

MVP完成後に追加予定の機能リスト。優先度順に記載。

---

## Priority 1: Monetization (収益化)

### 広告実装
- [ ] 全画面バナー広告（インタースティシャル）
  - トリガー: 3プレイ終了後 / ホームに戻る時
  - 初日〜2日目は薄く（定着優先）
- [ ] リワード広告
  - 失敗後: +1回復活（ミス1回無効）
  - クリア後: ボーナス分析表示
  - 今日の追加問題を解放
- [ ] バナー広告
  - Result画面下部

### IAP (アプリ内課金)
- [ ] 広告削除パック
- [ ] スキン課金
  - パネルデザイン: ネオン / ミニマル / 和風 / レトロ
  - タップSE変更
  - 背景テーマ: 昼 / 夜 / サイバー
- [ ] 分析レポート課金
  - 反応速度の推移（7日/30日）
  - ミス率ヒートマップ
  - タップ間隔の安定度
- [ ] コンテンツパック課金（Sentenceモード）
  - レベル別: 初級 / 中級 / 上級
  - テーマ別: 旅行 / 食べ物 / ことわざ / 雑学
  - 親子向け: 小学1〜2年セット

### サブスクリプション
- [ ] 日替わり問題（毎日3問追加）
- [ ] 週次リーグ参加（昇格/降格）
- [ ] 広告オフ
- [ ] 詳細分析フル解放
