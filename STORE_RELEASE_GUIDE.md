## 2. 必要アセットの作成

### 2.1 アプリアイコン

**仕様:**
- サイズ: 1024 x 1024 px
- フォーマット: PNG (透過なし、角丸なし)
- 背景: 単色または明確なデザイン

**デザインのコツ:**
```
推奨: シンプルで認識しやすいデザイン
- アプリ名の頭文字「M」をモチーフに
- ゲームのパネルをイメージした格子デザイン
- 背景色: #1E2A38 (ダークブルー - 現在のsplash色)
- アクセント: #2DD4BF (ティール) または #0D93F2 (ブルー)

NG例:
- 文字が小さすぎる
- 透過背景
- 写真そのまま
```

**作成ツール:**
- Figma (無料)
- Canva (無料)
- Adobe Illustrator

### 2.2 スクリーンショット

**必要サイズ:**

| デバイス | サイズ | 必須/任意 |
|----------|--------|-----------|
| iPhone 6.7" (14/15 Pro Max) | 1290 x 2796 | 必須 |
| iPhone 6.5" (11/12/13 Pro Max) | 1284 x 2778 | 必須 |
| iPhone 5.5" (8 Plus) | 1242 x 2208 | 必須 |
| iPad Pro 12.9" | 2048 x 2732 | 任意 |

**必要枚数:** 最低3枚、最大10枚

**推奨スクリーンショット構成:**
```
1枚目: ホーム画面 + キャッチコピー
   → 「あなたの脳の速度を測定」

2枚目: ゲームプレイ画面 (数字モード)
   → 「反射神経を鍛える」

3枚目: 結果画面
   → 「タイムを記録して成長を実感」

4枚目: モード選択
   → 「5つのゲームモード」

5枚目: ランキング/統計画面 (あれば)
   → 「自己ベストに挑戦」
```

**スクリーンショット作成ツール:**
- [Shots.so](https://shots.so) - 無料モックアップ
- [AppMockUp](https://app-mockup.com) - 無料
- Figma テンプレート

### 2.3 プライバシーポリシー

**必須記載事項:**
1. 収集するデータの種類
2. データの使用目的
3. 第三者への共有有無
4. ユーザーの権利
5. 連絡先

**簡易テンプレート (広告ありの場合):**

```markdown
# プライバシーポリシー

最終更新日: 2025年X月X日

## 1. 収集する情報

本アプリ「Mojic」は、以下の情報を収集する場合があります：

### 自動収集される情報
- デバイス識別子（広告ID）
- 利用状況データ（プレイ回数、スコア等）
- クラッシュレポート

### 広告について
本アプリはGoogle AdMobを使用して広告を表示します。
AdMobはパーソナライズド広告のためにデバイス情報を収集する場合があります。

## 2. 情報の使用目的

- アプリの機能提供
- 広告の表示
- アプリの改善

## 3. 第三者への情報共有

広告パートナー（Google AdMob）に対して、広告配信目的で情報が共有される場合があります。

## 4. お問い合わせ

ご質問がある場合は下記までご連絡ください：
メール: your-email@example.com
```

**ホスティング方法:**
1. GitHub Pages (無料)
2. Notion (無料で公開ページ作成可能)
3. 自社サイト

## 3. EASビルド設定

### 3.1 EAS CLIのインストール

```bash
npm install -g eas-cli
eas login
```

### 3.2 eas.json の作成

`rapid-type/eas.json`:
```json
{
  "cli": {
    "version": ">= 7.0.0"
  },
  "build": {
    "development": {
      "developmentClient": true,
      "distribution": "internal"
    },
    "preview": {
      "distribution": "internal",
      "ios": {
        "simulator": true
      }
    },
    "production": {
      "ios": {
        "buildConfiguration": "Release",
        "resourceClass": "m-medium"
      },
      "android": {
        "buildType": "app-bundle"
      }
    }
  },
  "submit": {
    "production": {
      "ios": {
        "appleId": "your-apple-id@example.com",
        "ascAppId": "YOUR_APP_STORE_CONNECT_APP_ID",
        "appleTeamId": "YOUR_TEAM_ID"
      }
    }
  }
}
```

### 3.3 app.json の更新

```json
{
  "expo": {
    "name": "Mojic",
    "slug": "mojic",
    "version": "1.0.0",
    "ios": {
      "bundleIdentifier": "com.mojic.app",
      "buildNumber": "1",
      "supportsTablet": true,
      "infoPlist": {
        "NSUserTrackingUsageDescription": "広告をパーソナライズし、より関連性の高いコンテンツを提供するために使用されます。",
        "ITSAppUsesNonExemptEncryption": false
      }
    },
    "plugins": [
      "expo-router",
      "expo-localization",
      [
        "expo-tracking-transparency",
        {
          "userTrackingPermission": "広告をパーソナライズし、より関連性の高いコンテンツを提供するために使用されます。"
        }
      ]
    ]
  }
}
```

### 3.4 ビルドコマンド

```bash
# プロジェクトをEASに登録
cd rapid-type
eas build:configure

# iOS用プロダクションビルド
eas build --platform ios --profile production

# App Store Connectに自動送信
eas submit --platform ios
```

## 4. App Store Connect設定

### 4.1 新規アプリの作成

1. [App Store Connect](https://appstoreconnect.apple.com) にログイン
2. 「マイApp」→「+」→「新規App」
3. 以下を入力:

| 項目 | 入力値 |
|------|--------|
| プラットフォーム | iOS |
| 名前 | Mojic - 脳トレタイピング |
| プライマリ言語 | 日本語 |
| バンドルID | com.mojic.app |
| SKU | mojic-app-001 |

### 4.2 アプリ情報タブ

#### カテゴリ設定
```
プライマリ: ゲーム > パズル
セカンダリ: 教育 > 教育ゲーム
```

#### コンテンツ権利
- 「このAppに第三者のコンテンツが含まれていることを証明する権利を持っています」をチェック

#### 年齢制限 (App Rating)
質問に正直に回答。ゲームアプリの場合、通常は「4+」になる。
- 暴力的コンテンツ: なし
- 性的コンテンツ: なし
- ギャンブル: なし

### 4.3 価格および配信状況

```
価格: 無料
配信地域: 全世界 (または日本のみで開始)
```

### 4.4 Appプライバシー

**収集データの申告 (広告ありの場合):**

| データタイプ | 収集 | 用途 |
|-------------|------|------|
| 広告ID | はい | サードパーティ広告 |
| 使用状況データ | はい | 分析 |
| クラッシュデータ | はい | アプリの機能 |

| 属性 | 無料, シンプル | 競合との差別化 |

**競合分析で見つけたキーワード:**
```
- シュルテテーブル (視野トレーニング)

- 動体視力 (スポーツ系ユーザー)
- IQテスト (知能系)
```

## 6. 審査対策 (リジェクト回避)

### 6.1 よくあるリジェクト理由と対策

| リジェクト理由 | 対策 |
|---------------|------|
| **Guideline 2.1** - アプリが不完全/クラッシュする | テストを徹底。全画面遷移確認 |
| **Guideline 2.3** - 説明と機能が一致しない | 説明文は正確に |
| **Guideline 4.2** - 最低限の機能がない | MVP以上の機能を実装 |
| **Guideline 5.1.1** - データ収集の説明不足 | プライバシーラベルを正確に |
| **Guideline 5.1.2** - ATT未対応 | 広告使用時は必須対応 |

### 6.2 ATT (App Tracking Transparency) 対応

iOS 14.5以降、広告IDを使用する場合は必須。

**インストール:**
```bash
npx expo install expo-tracking-transparency
```

**実装:**
```typescript
// App.tsx または起動時に呼び出し
import { requestTrackingPermissionsAsync } from 'expo-tracking-transparency';

useEffect(() => {
  (async () => {
    const { status } = await requestTrackingPermissionsAsync();
    if (status === 'granted') {
      // パーソナライズド広告を表示
    } else {
      // 非パーソナライズド広告を表示
    }
  })();
}, []);
```

### 6.3 審査用メモ (Review Notes)

```
【テストアカウント】
不要（ログイン機能なし）

【アプリの使い方】
1. ホーム画面でゲームモード (Numbers, Alphabet等) を選択
2. START GAMEボタンをタップ

【広告について】
Google AdMobを使用。ゲーム終了後にインタースティシャル広告を表示予定。
（初回審査時は広告なしでも可）

【その他】
シンプルな脳トレゲームです。個人情報の収集は行っていません。
```

```
□ スクリーンショットが実際の画面と一致

```

## 7. 申請後の対応

### 7.1 審査タイムライン

```
通常: 24-48時間
混雑時: 3-7日
初回申請: 長くなる傾向
```

### 7.2 リジェクト時の対応

1. **落ち着いて理由を確認**
   - Resolution Center で詳細確認

2. **対応方針**
   - 明確なガイドライン違反 → 修正して再申請
   - 不明確な場合 → 返信して詳細を確認

3. **再申請**
   - 修正内容を Review Notes に記載
   - buildNumber をインクリメント

### 7.3 承認後のタスク

```
□ リリース日を設定 (即時 or 指定日)
□ App Store ページの確認
□ ダウンロードテスト
□ SNSで告知
□ アップデート計画の策定
```
